## How to install and run
1. Install react at your local upon forking the repository

2. Navigate to the Project Directory:
```bash
cd your-forked-repo
```

3. Run the following commands
```bash
npm install
or
yarn install
```

3. Run the development server
```bash
npm start
or 
yarn start
```
